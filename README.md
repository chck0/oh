## 페르소나 (4/28 초안)

> 아래는 4/28 미팅에서 합의한 초안. 5월 1주차에 최종 확정 예정.

| 페르소나 | 레퍼런스 | 전문 영역 | 응답 예시 |
|---------|---------|----------|----------|
| 🎤 MC | 유재석 | 대화 진행, 중재, 요약 | "지금까지 나온 의견 정리해볼게요." |
| 📊 부동산 전문가 | 슈카 | 실거래가, 시장 동향, 지역 이슈, 미래 전망 | "이 동네 최근 3개월 실거래가 평균 7.2억. [출처: 국토교통부 API]" |
| 🌱 라이프 설계사 | 인생 선배 | 가치관, 선택 기준, 생애 라이프 스테이지 | "5년 뒤에도 이 동네에 살고 싶으세요?" |
| 🏦 Banker | 은행 담당자 | 대출 한도, DSR, 금리, 정부 지원 상품 | "디딤돌 대출 최대 4억까지 가능합니다. DSR 35% 이내." |
| 👋 선배 | 2달 전 첫 집 구매자 | 실제 구매 경험, 솔직한 후기 | "저도 그 고민 했어요. 등기 치고 나서 제일 놀랐던 게..." |
| 🏠 공인중개사 | 동네 중개사 | 공시지가, 실제 매물 정보 | "그 단지 실제로 나온 매물은 지금 2개고, 호가는..." |

### 페르소나 역할 경계 원칙

- **MC**: 대화 진행·중재·요약만. 직접 의견 제시 금지
- **부동산 전문가**: 데이터 기반 시장 정보만. 근거 없는 전망 금지
- **라이프 설계사**: 가치관·라이프 스테이지만. 재무 계산·시장 분석 금지
- **Banker**: 대출·금융 정보만. 집값 전망·라이프 조언 금지
- **선배**: 경험 공유만. 전문가적 분석·데이터 인용 금지
- **공인중개사**: 공시지가·매물 정보만. API 데이터 인용 금지

## 프로젝트 구조

```
├── agents/                    # 페르소나 명세서 (Markdown)
├── src/
│   ├── main.py                # CLI 엔트리포인트
│   ├── app.py                 # Streamlit Web UI
│   ├── meeting.py             # 회의 오케스트레이터
│   ├── personas.py            # 페르소나 로더 + 시스템 프롬프트
│   ├── real_estate.py         # 국토교통부 API + 샘플 데이터
│   ├── rent_vs_buy.py         # 전세 vs 매수 비교 분석기
│   └── demo_mock.py           # Mock 데모 (API 불필요)
├── tests/                     # pytest 테스트 스위트
├── meetings/                  # 회의록 저장 디렉토리
├── MANIFESTO.md               # 핵심 가치와 설계 원칙
└── glossary.md                # 용어집
```
