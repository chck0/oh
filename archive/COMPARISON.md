## ChatGPT 응답 (단일 AI)

```
광명시 광명동 일대는 7호선 환승과 1호선 접근성 덕분에 광화문 출근이
가능한 권역으로, 24평형 5.3억은 최근 시세를 감안하면 무리한 가격은
아닙니다. 다만 금리 인상이 지속되는 환경이라 대출 부담을 한 번
점검해 보시고, 학군과 주변 인프라도 직접 둘러보시는 것을 권장합니다.
인근에 재건축 이슈가 있는 단지들이 있어 향후 가치 상승 여지도

```

**문제점:**
- ❌ "무리한 가격은 아닙니다" → 근거 없음. 실거래 P50 비교 0건
- ❌ "재건축 이슈" → 출처·발표일·실현 일정 없음 (단순 추측)
- ❌ "금리 인상" → 구체적 LTV·DSR·월 상환 계산 없음
- ❌ 통근 시간 구체 측정 없음 (광명 → 광화문 실제 몇 분?)
- ❌ 단지 노후도·대수선 이력 등 단지 리스크 검증 없음
- ❌ 한 화면 종합 결론 없음 — 결정에 도움이 안 됨
- ❌ 5명이 다른 시각으로 갈리지 않음 — Second Opinion 부재

## 차별점 비교표

| 항목 | ChatGPT | 우리 시스템 |
|---|---|---|
| **데이터 출처** | 없음 ("무리한 가격은 아닙니다") | 국토부 실거래가 + Kakao Local + 한국부동산원 + 금융위 모든 수치에 [출처] 명시 |
| **컨셉 포지셔닝** | 친근 자문 ("권장합니다") | **검증 (Second Opinion)** — "사세요/사지 마세요"가 아닌 다관점 검증 |
| **다관점 분석** | 단일 시각 + 긍정+부정 모호 혼합 | 5인 영역별 독립 분석 (시세·입지·리스크·재무·미래가치) |
| **반박·리스크** | "다만~" 수준 | 각 분석가가 **반드시 1개 이상 의심·반박·리스크** 제시 |
| **별점 시각화** | 없음 | 5개 카테고리 별점 + 종합 평점 + 합의 결론 한 줄 |
| **재무 계산** | 추상적 조언만 | 보유자산·LTV·DSR·취득세·월 상환 BuyerProfile 기반 계산 |
| **호가 적정성** | 없음 | 동일 단지·평형 P50 vs 호가 비교 |
| **5~10년 미래가치** | 단순 추측 | 호재 출처·발표일·실현 일정 + 악재 균형 검증 |
| **결정 책임** | 모호 | "결정은 사용자가" — 명시적 강매 금지 |
| **후속 액션** | 없음 (대화 종료 시 사라짐) | 드릴다운·대화·PDF 3가지 가안 |

## 차별점 3줄 요약 (발표용)

1. **5인 영역 분리 + 별점 종합** — ChatGPT는 한 명이 모호하게 답하지만, 우리는 시세·입지·리스크·재무·미래가치 5명이 자기 영역에서만 검증해 별점이 갈린다. 별점 흩어짐이 곧 트레이드오프 시각화다.

2. **검증·반박 강제 + 출처 의무** — ChatGPT는 "권장합니다"로 끝나지만, 우리는 각 분석가가 반드시 1개 이상의 의심·반박·리스크를 제시한다. 모든 수치에는 국토부·한국부동산원·금융위 등 출처가 자동으로 따라붙는다.

3. **결정을 대신하지 않는다** — ChatGPT는 "이렇게 보입니다"로 모호하게 권유하지만, 우리는 "5명 중 X명이 신중 검토 권합니다 + 핵심 쟁점은 A vs B"로 결정의 재료만 깔끔하게 한 페이지로 보여주고, 마지막 결정은 사용자에게 돌려준다.

## 발표 시 대응 스크립트

**질문**: "이거 ChatGPT에 5명 역할 나눠서 물어보면 되지 않나요?"

**답변**:
> "좋은 질문입니다. 실제로 해봤습니다. (비교 화면 보여주며) ChatGPT에 같은 매물을 물어보면 '무리한 가격은 아닙니다, 다만 금리 환경 점검해 보세요' 정도의 모호한 답변이 옵니다. 출처가 없고, 5명이 다른 시각으로 갈리지 않고, 별점·합의 결론 같은 결정 도구가 없습니다.
>
> 저희 시스템은 세 가지가 다릅니다. 첫째, 시세·입지·리스크·재무·미래가치 5명이 영역을 침범하지 않고 각자 자기 영역에서만 검증합니다. 둘째, 모든 수치에 국토부·한국부동산원 같은 출처가 자동으로 따라붙고, 각 분석가는 반드시 1개 이상의 의심·반박·리스크를 제시하도록 강제됩니다 — 5명이 다 좋다고 하면 시스템 실패입니다. 셋째, 종합 리포트가 한 페이지에 별점·합의·핵심 쟁점·후속 액션을 다 담아서, 사용자가 그 트레이드오프를 보고 결정하게 됩니다.
>
> 우리는 검증 시스템이라 '사세요/사지 마세요'를 말하지 않습니다. 어제 몰랐던 것을 오늘 알게 해드릴 뿐입니다. 결정은 사용자가 합니다."
