# 부동산 시세분석 서비스 비교 (초보자 가이드)

> 네이버부동산 / 호갱노노 / 아실이 시세를 분석하는 방법

---

## 🏢 네이버부동산

**핵심 개념: "지금 얼마에 팔고 있나?"**

```
시세 = 현재 매물 호가 + 실거래가 평균
```

### 분석 방식

```
1단계: 매물 수집
   └─ 공인중개사가 직접 등록한 매물 가격(호가) 수집

2단계: 실거래가 비교
   └─ 국토교통부 실거래가 API 연동
   └─ "이 아파트 최근 3개월 실제 거래된 가격"

3단계: 시세 범위 산출
   └─ 하한가 ~ 상한가 범위로 표시
   └─ ex) "이 단지 33평 → 8억 ~ 9억 2천"
```

### 보여주는 정보

| 항목 | 설명 |
|---|---|
| 호가 | 집주인이 "이 가격에 팔겠다"고 올린 금액 |
| 실거래가 | 실제로 계약서 쓴 금액 (국토부 공개) |
| 매물 수 | 현재 나와있는 매물 개수 |
| 시세 그래프 | 월별 평균 거래가 추이 |

> **초보자 포인트:** 호가 ≠ 실거래가. 네이버는 이 둘의 차이를 보여주는 것이 핵심

---

## 🏠 호갱노노

**핵심 개념: "이 가격이 비싼 건지 싼 건지?"**

```
시세분석 = 실거래가 흐름 + 같은 단지 내 층/향 비교
```

### 분석 방식

```
1단계: 실거래가 전수 수집
   └─ 국토부 데이터를 아파트 동·호수 단위까지 분해

2단계: 같은 면적 내 비교
   └─ "103동 저층 vs 고층 가격 차이"
   └─ "같은 평형인데 향(남향/북향)에 따른 가격 차이"

3단계: 시계열 그래프
   └─ 점(dot)으로 실거래 하나하나 표시
   └─ 언제 비쌌고 언제 쌌는지 한눈에 파악

4단계: 고점 대비 현재
   └─ "고점 대비 -15% 하락한 상태" 표시
```

### 보여주는 정보

| 항목 | 설명 |
|---|---|
| 실거래 산점도 | 점 하나 = 실제 거래 1건 |
| 고점/저점 표시 | 역대 최고가, 최저가 기준선 |
| 면적별 가격 분포 | 같은 단지 내 평형간 가격 차이 |
| 학군 정보 | 배정 학교 + 학업 성취도 |

> **초보자 포인트:** "지금 이 가격이 역사적으로 비싼가 싼가"를 보는 데 특화

## 📊 아실 (아파트 실거래가)

**핵심 개념: "수요와 공급으로 가격 방향을 예측한다"**

```
시세분석 = 매물 증감 + 거래량 + 입주물량 → 가격 방향 예측
```

### 분석 방식

```
1단계: 매물 변화량 추적
   └─ 매물이 늘어남 → 공급 증가 → 가격 하락 압력
   └─ 매물이 줄어듦 → 공급 감소 → 가격 상승 압력

2단계: 거래량 분석
   └─ 거래량 급증 = 실수요 확인 = 가격 상승 선행지표
   └─ 거래량 급감 = 관망세 = 가격 조정 가능성

3단계: 입주물량 확인
   └─ 향후 2년 새 아파트 공급 예정량
   └─ 공급 많음 → 기존 아파트 가격 압박

4단계: 외지인 거래 비율
   └─ 타 지역 사람이 많이 사면 → 투자 수요 유입 신호
```

### 보여주는 정보

| 항목 | 설명 |
|---|---|
| 매물 증감 차트 | 이번 주 매물이 늘었나 줄었나 |
| 거래량 히트맵 | 지역별 거래 활발 정도를 색으로 표시 |
| 입주물량 달력 | 언제 얼마나 새 아파트가 공급되는지 |
| 외지인 매수 비율 | 투자 수요 vs 실수요 비율 |

> **초보자 포인트:** "앞으로 가격이 오를지 내릴지"의 선행지표에 집중

---

## 세 서비스 한눈에 비교

```
네이버부동산 ──── "지금 얼마야?"        → 현재 가격 확인
호갱노노     ──── "비싼 거야 싼 거야?"  → 역사적 위치 확인
아실         ──── "앞으로 어떻게 돼?"   → 미래 방향 예측
```

| | 네이버부동산 | 호갱노노 | 아실 |
|---|---|---|---|
| 주요 데이터 | 호가 + 실거래가 | 실거래가 분석 | 수급 지표 |
| 강점 | 매물 현황 | 가격 히스토리 | 시장 방향성 |
| 약점 | 방향성 없음 | 수급 분석 없음 | 개별 단지 약함 |
| 초보 활용법 | 매물 찾기 | 가격 판단 | 타이밍 판단 |

---

## 실전 활용 순서

```
STEP 1. 아실       → 지금 이 지역 사야 할 타이밍인가?
STEP 2. 호갱노노   → 이 단지 가격이 비싼가 싼가?
STEP 3. 네이버부동산 → 실제 매물 찾고 호가 확인
```
