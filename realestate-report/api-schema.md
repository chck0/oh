## 파이프라인 설계 시 주의사항

### 1. 계약 해제 건 반드시 제외
```python
# cdealType이 공백이 아닌 거래 = 취소된 거래
items = [i for i in items if not i.findtext('cdealType', '').strip()]
```

### 2. 지분거래 제외 권장
```python
items = [i for i in items if i.findtext('landLeaseholdGbn', '').strip() == 'N']
```

### 3. 거래금액 파싱 (쉼표 제거 필요)
```python
price = int(item.findtext('dealAmount', '0').replace(',', '').strip())
```

### 4. 평단가 계산
```python
pyeong = area / 3.3058
price_per_pyeong = price / pyeong
```
