# ChatGPT vs 우리 시스템 비교 시연 자료

> PREMORTEM 시나리오 6 대응: "ChatGPT랑 뭐가 달라요?"
> 같은 질문을 ChatGPT에 물어본 응답 vs 우리 시스템의 3인 자문을 나란히 비교

---

## 질문: "판교 출근인데 마포구 아파트 84㎡ 호가 13억 8천만원, 적정한가요?"

### ChatGPT 응답 (단일 AI)

```
마포구 아파트 84㎡ 호가 13억 8천만원은 최근 시장 상황을 감안하면
적정 수준으로 보입니다. 마포구는 교통이 편리하고 인프라가 잘 갖춰져 있어
수요가 꾸준한 편입니다. 다만 최근 금리 인상으로 매수세가 다소 위축된
측면이 있어 협상 여지가 있을 수 있습니다. 판교 출근이라면 출퇴근 시간도
미리 확인해 보시는 걸 권장합니다.
```

**문제점:**
- ❌ "적정 수준으로 보입니다" → 근거 없음. 실거래 데이터 0건
- ❌ "수요가 꾸준한 편" → 출처 없는 추측
- ❌ 국토교통부 실거래 P50 비교 없음
- ❌ DSR·대출한도·총취득비용 계산 없음
- ❌ 판교 출근 기준 통근 적합성 구체 분석 없음
- ❌ 내 예산·자기자본·가족 구성에 맞는 자문 없음
- ❌ 대화 종료 후 정리·넥스트 액션 없음

---

### 우리 시스템 응답 (3인 전문가 자문)

> MC 인터뷰어가 먼저 출근지(판교), 예산(6억), 자기자본(2억), 선호 지역(마포구)을 수집한 뒤
> 3인 전문가에게 BuyerProfile을 전달합니다.

#### 🏠 중개사 (입지·학군·교통)
```
판교 출근이라면 신분당선 라인을 먼저 보세요. 마포구에서 판교까지
대중교통 70~80분 구간인데, 같은 예산이면 수지·분당 신분당선 역세권에서
더 가까운 거리로 84㎡를 살 수 있습니다. 마포구는 합정·공덕 인근
역세권 단지가 학군·인프라 면에서 강점이 있으니, 판교 출퇴근을
감수할 만한 이유가 있는지 먼저 정하시는 게 순서입니다.
```

#### 💰 재무설계사 (DSR·대출·총취득비용)
```
예산 6억, 자기자본 2억 기준으로 봅시다. DSR 40% 기준 연소득을
알아야 정확하지만, 대출 4억을 가정하면 30년 고정 4% 기준 월 190만원
수준입니다. 여기에 취득세 (13억 8천만원 × 1.1%) 약 1,518만원,
중개보수 (최대 0.7%) 약 966만원, 이사비 포함 총취득비용은 2.5억
이상 예상됩니다 [출처: 지방세법 시행령, 2024 기준]. 자기자본 2억으로
취득세·이사비를 먼저 충당하고 나면 실제 자기자본은 1.5억 미만—
대출이 4억을 초과할 수 있습니다.
```

#### 📊 시장분석가 (실거래 P50·공급리스크)
```
마포구 84㎡ 최근 3개월 실거래 P50은 13억 2,000만원입니다
[출처: 국토교통부 실거래가 공개시스템, API: getRTMSDataSvcAptTrade,
LAWD_CD: 11440, DEAL_YMD: 202602/202601/202512, N=7건, 조회일: 2026-04-30].
호가 13억 8천만원은 P50 대비 +4.5%로 '적정' 범위(±15%) 안에 있습니다.
단, 반론을 드립니다: 마포구 2026년 입주 예정 물량이 1,200세대로
전년 대비 40% 증가 구간입니다. 공급 압력이 3개월 이상 지속되면
P50이 하향 조정될 수 있어 지금 호가가 6개월 뒤에는 고평가로
뒤바뀔 수 있습니다.
```

#### 📝 비서실장 (자동 상담록)
```
## 👤 고객님 구매 조건
- 출근지: 판교 / 예산: 6억 / 자기자본: 2억 / 선호: 마포구 84㎡

## 에이전트별 핵심 의견
- 🏠 중개사: 판교 출퇴근 70~80분 감안 시 수지·분당 신분당선 대안 검토 권장
- 💰 재무설계사: 총취득비용 2.5억+ 고려 시 실자기자본 부족—DSR 한도 재확인 필요
- 📊 시장분석가: 호가 P50 대비 +4.5% 적정, 단 입주 물량 증가 리스크 주의

## ✅ 넥스트 액션
1. 은행 DSR 사전심사 예약 (KB·신한 각 1곳, 다음 주까지, 담당: 본인)
2. 마포구 합정·공덕 vs 수지 신분당선 역세권 임장 비교 (이번 주 토요일)
3. 마포구 입주 물량 추이 재확인 후 관망 여부 결정 (2주 내)
```

## 차별점 비교표

| 항목 | ChatGPT | 우리 시스템 |
|-|-|-|
| **데이터 출처** | 없음 ("적정 수준으로 보입니다") | 국토교통부 실거래가 API 직접 연동 + 모든 수치에 [출처] 명시 |
| **맥락 수집** | 질문 그대로 답변 | MC 인터뷰어가 BuyerProfile 먼저 수집 (출근지·예산·자기자본·가족) |
| **관점 다양성** | 단일 관점, 긍정+부정 모호하게 혼합 | 중개사(입지) / 재무설계사(재무) / 시장분석가(데이터) 3인 역할별 전문 분석 |
| **리스크 분석** | "다만~" 수준의 언급 | 시장분석가가 구조적으로 최소 1개 하방 리스크 반드시 제기 |
| **재무 계산** | 추상적 조언만 | DSR·대출한도·취득세·총취득비용 BuyerProfile 기반 계산 |
| **호가 적정성** | 없음 | 실거래 P50 대비 호가 ±% 비교 + 적정/고평가/저평가 판정 |
| **대안 비교** | "판교 출퇴근 확인해보세요" | 수지·분당 신분당선 등 구체 대안 지역 제시 |
| **정리·후속** | 없음 (대화 종료 시 사라짐) | 비서실장이 넥스트 액션(동사+담당자+기한) 자동 생성 |

---

## 차별점 3줄 요약 (발표용)

1. **국토부 실거래 P50 직접 연동 + 출처 명시** — ChatGPT는 "적정 수준으로 보입니다"이지만, 우리는 국토교통부 API에서 가져온 실거래 중앙값(P50)에 [LAWD_CD, N=건수, 조회일]을 달아 신뢰할 수 있는 숫자만 말한다.

2. **MC 인터뷰로 BuyerProfile 수집 후 맞춤 자문** — ChatGPT는 맥락 없이 답하지만, 우리는 MC 인터뷰어가 먼저 출근지·예산·자기자본·가족 구성을 파악한 뒤 중개사·재무설계사·시장분석가 3인이 그 맥락에 맞춰 각자 영역에서 전문적으로 발언한다.

3. **비서실장 자동 상담록** — ChatGPT는 대화가 끝나면 수다로 끝나지만, 우리는 비서실장이 결정사항과 동사+담당자+기한이 담긴 넥스트 액션을 자동으로 정리해 실제 내 집 마련 행동으로 이어진다.

## 발표 시 대응 스크립트

**질문**: "이거 ChatGPT에 역할 나눠서 물어보면 되지 않나요?"

**답변**:
> "좋은 질문입니다. 실제로 해봤습니다. (비교표 보여주며) ChatGPT에 같은 질문을 하면 '적정 수준으로 보입니다, 다만 금리가 올랐으니 협상 여지가 있을 수 있습니다' 이렇게 답합니다. 출처가 없고, 실거래 데이터가 없고, 내 예산과 자기자본에 맞는 계산이 없습니다.
>
> 저희 시스템은 세 가지가 다릅니다. 첫째, 국토교통부 실거래가 API에서 P50을 가져와 호가와 비교하고 모든 수치에 출처를 답니다. 둘째, MC 인터뷰어가 먼저 제 상황을 파악한 뒤 중개사·재무설계사·시장분석가 세 명이 각자 영역에서 전문적으로 분석합니다. 특히 시장분석가는 반드시 하방 리스크를 지적하도록 설계해서 확증편향을 방지합니다. 셋째, 상담이 끝나면 비서실장이 넥스트 액션을 자동으로 정리해줍니다. ChatGPT는 좋은 대화를 하지만 결정을 도와주진 않습니다."
