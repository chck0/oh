# 부동산 시세분석 서비스 발전 방향

> 네이버부동산 / 호갱노노 / 아실의 한계를 넘어선 추가 기능 제안

---

## 기존 서비스의 공통 한계

```
기존 서비스의 한계
├─ 데이터 보여주기에 집중 → "해석"은 사용자가 직접
├─ 과거/현재 중심 → 미래 예측 약함
├─ 아파트 단지 중심 → 개인 상황 반영 없음
└─ 정보 파편화 → 여러 사이트 돌아다녀야 함
```

---

## 추가 기능 5가지

### 1. AI 자동 감정평가 (AVM)

> 현재: 실거래가 나열 → 사용자가 직접 "비싼지 싼지" 판단
> 추가: AI가 적정가를 계산해서 알려줌

```
적정가 = 모델(면적, 층, 향, 학군, 역거리, 최근거래, 금리, 매물수 ...)

예시 출력:
┌─────────────────────────────────────┐
│ 이 매물 호가: 9억 2천               │
│ AI 산출 적정가: 8억 4천 ~ 8억 9천  │
│ 판정: ⚠️ 호가가 적정가보다 높음     │
└─────────────────────────────────────┘
```

**차별점:** 지금은 사용자가 여러 사이트 돌아다니며 직접 판단, AI가 "이 가격 괜찮음/비쌈/저렴"을 즉시 판정해줌

---

### 2. 내 상황 맞춤 금융 시뮬레이터

> 현재: 단순 대출 계산기 수준
> 추가: 내 소득/자산/생애주기를 넣으면 전략 추천

```
입력값
├─ 내 연소득: 6,000만원
├─ 보유 현금: 2억
├─ 거주 기간 목표: 5년
└─ 결혼 예정: 2년 후

출력값
├─ 매수 가능 최대가: 7억 3천 (DSR 40% 기준)
├─ 월 예상 상환액: 178만원
├─ 5년 후 예상 총비용: 전세 vs 매수 비교
└─ 추천 전략: "전세 2년 후 매수 타이밍이 유리"
```

**차별점:** 현재 서비스는 "시장 데이터"만 제공, 이건 "내 상황에 맞는 답"을 줌

---

### 3. 법적 리스크 자동 분석

> 현재: 등기부등본은 사용자가 직접 열람/해석
> 추가: 등기 데이터 파싱 후 리스크 자동 감지

```
자동 감지 항목
├─ 근저당 설정 금액 / 선순위 채권 합계
├─ 압류·가처분·가등기 여부
├─ 전세 세입자 수 및 보증금 합계
└─ "이 집 경매 넘어가면 내 전세금 돌려받을 수 있나?"

예시 출력:
┌──────────────────────────────────────────┐
│ 🔴 위험: 선순위 채권 합계 6억 2천        │
│    집값 대비 채권 비율 82%               │
│    → 전세 계약 시 보증금 회수 위험       │
└──────────────────────────────────────────┘
```

**차별점:** 전세사기 예방에 직결, 현재 어떤 서비스도 자동화하지 않은 영역

---

### 4. 생활권 품질 종합 스코어

> 현재: 학군 정보 + 역 거리 나열
> 추가: 내 라이프스타일 기준으로 점수화

```
사용자 설정: "맞벌이 부부, 자녀 없음, 재택근무 3일"

단지 스코어 (100점 만점)
├─ 편의시설: 88점 (마트, 카페, 병원 반경 500m)
├─ 조용함:   72점 (소음지도, 유흥가 거리)
├─ 대기질:   65점 (공기질 측정 데이터)
├─ 치안:     81점 (범죄 발생 통계)
└─ 종합: 76점 / 이 지역 상위 23%
```

**차별점:** 지금은 모든 사람에게 같은 정보 제공, 이건 "나한테 좋은 동네"를 찾아줌

---

### 5. 매수/매도 타이밍 알림 에이전트

> 현재: 사용자가 매일 앱 들어가서 확인
> 추가: AI가 조건 충족 시 능동적으로 알림 발송

```
사용자 설정
├─ 관심 단지: 마포래미안푸르지오 84㎡
├─ 목표 매수가: 12억 이하
└─ 조건: 매물 3개 이상 + 전월 대비 거래량 증가

AI 모니터링 항목
├─ 매물 가격 변동 감지
├─ 급매 출현 감지 (평균가 대비 5% 이상 저렴)
├─ 거래량 이상 급증 감지
└─ 정책 변경 감지 (규제지역 해제 등)

→ 조건 충족 시 카카오톡/문자 즉시 알림
```

**차별점:** 현재 알림은 단순 가격 알림, 이건 복합 조건 기반 인텔리전트 알림

---

## 종합 포지셔닝

```
기존 서비스       →  "데이터 제공"
추가할 서비스     →  "판단과 행동까지 도와주는 부동산 어드바이저"
```

| 추가 기능 | 해결하는 문제 | 개발 난이도 |
|---|---|---|
| AI 자동 감정평가 | "이 가격 적당해?" 판단 | ★★★★☆ |
| 맞춤 금융 시뮬레이터 | "내가 살 수 있어?" 전략 | ★★★☆☆ |
| 법적 리스크 분석 | 전세사기/사기 예방 | ★★★★☆ |
| 생활권 스코어 | "나한테 좋은 동네?" | ★★★☆☆ |
| 타이밍 알림 에이전트 | 매일 확인하는 수고 제거 | ★★★★★ |

---

## 구현 우선순위 추천

```
1순위  법적 리스크 분석     → 전세사기 이슈로 사회적 수요 높음
2순위  맞춤 금융 시뮬레이터 → 개발 난이도 낮고 사용자 체감 효과 큼
3순위  AI 자동 감정평가     → 핵심 차별화 포인트, 데이터 축적 필요
4순위  타이밍 알림 에이전트 → 리텐션(재방문) 극대화
5순위  생활권 스코어        → 외부 데이터 연동 많아 후순위
```
