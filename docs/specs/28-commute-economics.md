## 1. Why (차별화 포인트)

호갱노노·네이버부동산은 "이 아파트가 얼마인가"를 보여준다.
BADUGI는 "이 아파트로 이사하면 내 삶이 어떻게 바뀌는가"를 보여준다.

통근시간 수치(30분)는 추상적이다.
"이 아파트 선택 시 연간 240시간을 통근에 씁니다"는 구체적이다.
"월 교통비 약 6만원, 더 가까운 단지 대비 연 48시간 절약"은 의사결정 근거가 된다.

---

## 2. Scope

### In-scope
- 추천 카드 + 리스트 카드에 월 교통비·연간 통근 시간 표시
- 같은 버킷 내 최단 단지 대비 연간 시간 차이 표시
- 수도권 교통카드 기준 편도 요금 근사 계산

### Out-of-scope
- 자가용 연료비 계산
- 개인 할인카드 반영 (정기권·청년 할인)
- 지역별 요금 차이 (수도권 외)

## 3. 요금 계산 공식

```python
# 수도권 교통카드 편도 요금 근사 (2024년 기준)
def _commute_fare(total_time_min: int) -> int:
    if total_time_min <= 25:  return 1500   # 단거리 1회
    if total_time_min <= 40:  return 1800   # 중거리 or 환승
    if total_time_min <= 55:  return 2100   # 장거리 or 2회 환승
    return 2300                              # 60분+

monthly_cost  = fare × 2 × 20     # 왕복 × 월 20일
annual_hours  = time × 2 × 20 × 12 ÷ 60  # 왕복 × 연간(분→시간)
```
