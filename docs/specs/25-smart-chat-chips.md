# Spec 25: 스마트 초기 Chip — Claude가 답할 수 있는 질문으로 교체

> **상태**: In Progress 🔨
> **작성일**: 2026-05-30
> **구현 브랜치**: hjkang83

---

## 1. Why

기존 초기 Chip 5개 중 2개("지금 호가가 얼마야?", "전세가는 어떻게 돼?")는 실시간 데이터가 없어
Claude가 "확인 필요"만 반복하는 무의미한 질문임. 
우리가 보유한 데이터(실거래 통계, 시세 트렌드, POI, 건물정보, 웹검색)로 실제로 답변 가능한
질문으로 교체해 UX 품질을 높인다.

---

## 2. Scope

### In-scope
- `CHAT_SUGGESTIONS` 상수 교체 (result.html)
- Claude 답변 가능 여부 기준으로 질문 선정

### Out-of-scope
- 단지별 맞춤 초기 Chip (동적 생성은 F2 응답 후 이미 구현됨)

---

## 3. 새 Chip 구성

| 질문 | 근거 데이터 |
|------|------------|
| `최근 시세 트렌드 어때?` | trade_history 1년 통계 + 6개월 변동률 |
| `주변 단지 대비 시세는?` | 동 평균 시세 비교 |
| `단지 장단점 솔직하게 말해봐` | 건물정보 + POI |
| `투자 가치 있어?` | 시세 트렌드 분석 |
| `학군이랑 주변 환경은?` | DuckDuckGo 웹검색 |
