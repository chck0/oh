## 줄기 1: 재무적으로 검증해야 한다 → 재무설계사

### 위에서 아래로 읽기

```
혼자서도 내 집 마련 결정을 내릴 수 있다
    ↓ 그러려면?
재무적으로 검증해야 한다 (DSR, 대출 한도, 총취득비용, 월 상환액)
    ↓ 왜 지금은 안 되나?
DSR 40% 규제 하에서 내 소득으로 얼마까지 빌릴 수 있는지,
취득세·중개보수·이사비 포함 총취득비용이 얼마인지,
매달 갚을 수 있는 금액 범위가 어디까지인지
직접 계산하기가 어렵다.
ChatGPT에 물어봐도 "대략적으로" 말할 뿐
내 소득·자기자본 기반 정확한 계산은 안 해준다.
    ↓ 그래서 필요한 것은?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[기능] 재무설계사 에이전트
 - DSR 40% 기준 최대 대출 한도 계산
 - 자기자본 + 대출 = 총 구매 가능 예산
 - 취득세·중개보수·이사비 포함 총취득비용 산출
 - 모든 수치에 출처 명시
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

| 판단 | 결론 |
|-|-|
| 이 기능 없으면? | 근거 없는 예산 계획으로 내 집 마련 실패 → 핵심 가치 위반 |
| 중요도 | **필수** |

## 결론: 3단계 플로우가 Why Tree를 더 선명하게 만든다

기존의 "단일 입력 → C-suite 토론"은 맥락 수집 없이 자문이 시작됐다. 3단계 플로우로 바꾸니:

* **Stage 1 (MC 인터뷰)** → BuyerProfile 수집 → "내 상황"이라는 맥락 확보
* **Stage 2 (3인 자문)** → 재무설계사·시장분석가·중개사가 BuyerProfile 기반 맞춤 자문
* **Stage 3 (호가 적정성)** → 실거래 P50 vs 호가 비교 → 최종 매입 판단

기능을 추가하고 싶을 때의 질문:

> "이 기능에서 위로 올라갔을 때, 세 줄기 중 하나를 거쳐 핵심 가치에 도달하는가?"

도달하면 만든다. 도달하지 않으면 만들지 않는다.
