## 2. 페르소나 스토리 — 9년의 분투

민준은 2018년 29세에 서울 사무직 평균 대기업에 신입 입사. 입사 직후 자취 시작 (보증금 2,000만 + 월세 80만). 사원 3년차에 대리 진급, 8년차에 과장 진급.

매달 평균 280만 원 내외를 저축하며 *9년 누적 약 1.68억 원*. 여기에 야근수당·청약 이자·소액 투자 수익을 더해 *2억 원* 도달.

**이 2억 원은 단순한 통장 잔고가 아님.** 20대 후반부터 30대 중반까지의 *모든 사회생활 경험과 인내*를 치환한 결과물. 이 자본을 잃는 것은 *지난 10년에 대한 부정*으로 이어짐. 작은 리스크에도 민감하게 반응하는 심리 기제의 근원.

> *"인생에서 가장 큰 거래인데, 한두 사람 말만 듣고 결정할 수는 없다."*

## 3. 핵심 페인포인트

### 3.1. 정보 과잉의 역설 — *누구 말을 믿어야 할지 모름*

> 퇴근 후 유튜브를 켠다. 한 영상은 *"지금이 바닥, 무조건 사라"*고 외치고, 다음 영상은 *"2027년까지 더 빠진다"*고 한다. 댓글창은 양쪽 다 *"감사합니다 선생님"*.

- **유튜브 양극단**: 매수 강권 vs 하락 경고
- **동기 의견 충돌**: *"강남 불패"* vs *"경기도 신축이 답"*
- **부모님 추측**: *"친구한테 들었는데 어디가 뜬다더라"*

→ 인지적 마비. 결정 미루거나 충동 매수 위험.

### 3.2. 영업 멘트와의 신뢰 단절

> 부동산 사장: *"총각, 이거 진짜 급매야 급매. 어제 한 건 더 비싸게 나갔어. 지금 안 사면 다음 달엔 또 올라요."*

- 중개인 수수료 구조 → 단점 축소·장점 부각
- 모든 부동산이 *"지금이 기회다"* 화법
- *알아야 할 권리 박탈감* + *눈뜬장님 공포*

→ 중개인 주관이 배제된 *제3자 객관 검증* 갈구.

### 3.3. 매물 가치 검증의 어려움

| 검증 영역 | 페인포인트 |
|---|---|
| 호가 적정성 | "상꼭대기에서 물리는" 공포. 실거래가 시차·세부 조건 미반영 |
| 입지 진실성 | "도보 역세권"·"명품 학군" 미사여구 vs 실제 도보 시간·통학로 |
| 미래 성장성 | 5~10년 후 가치 — 인구·정책·호재 변동성 |

### 3.4. 알 권리 사각지대 — *3대 침묵 리스크*

중개인이 의도적 누락하는 정보:

| 리스크 | 영향 |
|---|---|
| **층간소음·이웃 분쟁** | 경찰 신고·고소·고발로 이어질 수 있는 생존 문제. 중개인 *"보통 수준"* 화법에 가려짐 |
| **고질적 내부 결함** | 장판 아래 곰팡이, 붙박이장 안쪽 누수, 보일러 노후, 수압 저하 |
| **인근 혐오·유해시설** | 쓰레기처리장·장례식장 입주 계획. 호재 대비 축소 보고 |

### 3.5. 시장 환경 불안

| 요소 | 페르소나에 미치는 영향 |
|---|---|
| **2026 서울 입주물량 -31.6%** (29,161가구) | "공급 절벽" FOMO 강화. *"지금 안 사면 영영 진입 못함"* |
| **GTX 지연** (A노선 발표→개통 10년+) | 미래가치 호재 신뢰성 훼손 |
| **3기 신도시 사전청약 취소** (인천 가정2지구·파주 운정3지구) | "정부 약속의 집"보다 *"실재하는 매물"* 집착 |
| **인구 절벽 공포** | "5년 뒤 매수 수요 감소하면 가치 하락?" |

## 4. 심리 기제 — *왜* 이렇게 불안해하는가

### 4.1. *단 한 번의 기회* 인식

2억 시드머니는 9년 노동의 치환물. **실패 시 회복 어려움**. 작은 리스크에도 강하게 반응.

### 4.2. *정보 비대칭이 가장 심한 시점*

생애 첫 구매 → 부동산 시장 경험 부재 → 어디부터 검증해야 할지 모름.

### 4.3. *결정 회피·과도한 검색* 사이클

확신 없는 상태에서 정보 더 찾을수록 양극단 의견에 노출 → 더 혼란. 무한 루프.

### 4.4. *제3자 객관성 갈구*

중개인·지인 모두 *편향*되어 있음을 인지. *데이터 기반 객관 분석*이 진짜 갈증.

## 5. 시스템이 다루는 페인포인트 vs 다루지 않는 페인포인트

| 페인포인트 | 시스템 처리 |
|---|---|
| 호가 vs 실거래가 간극 | ✅ 시세 카드 (회귀 잔차) |
| 영업 멘트 vs 실제 도보 | ✅ 입지 카드 (객관 거리) |
| 도보 역세권·학군 검증 | ✅ 입지 + 단지별 도보 접근성 (한국부동산원 RAG) |
| 단지 노후도·관리 품질 | ✅ 리스크 카드 |
| 미세먼지 국지 영향 (공장 등) | ✅ 리스크 카드 (시 평균 대비 상대값 + 트리거 시 발생원) |
| 교통 소음 (간선도로 인접) | ✅ 리스크 카드 (도로 거리 프록시) |
| 공급 절벽 FOMO | ✅ 미래가치 카드 (청약·정비사업 데이터) |
| 호재 의존도 (GTX 등) | ✅ 미래가치 카드 (회귀 잔차 + 이벤트 스터디) |
| 인구·세대 추세 | ✅ 미래가치 카드 (KOSIS·행안부) |
| **매물 호 단위 층간소음·누수·곰팡이** | ❌ 시스템 범위 밖 (임장 위임) |
| **권리관계 (등기·근저당·가압류)** | ❌ 시스템 범위 밖 (인터넷등기소 700원 직접 발급) |
| **자산·대출·DSR 시뮬레이션** | ❌ 시스템 범위 밖 (개인정보 민감) |
| **호재 실현 예언** | ❌ 시스템 범위 밖 (호재 의존도 분해까지만) |
| **사전청약 매물 검증** | ❌ 시스템 범위 밖 (매수 시나리오만) |
| **민간 플랫폼 정성 시그널** (직방·호갱노노 리뷰) | ❌ 시스템 범위 밖 (공공 데이터 약속) |

## 6. 사용자 시나리오 — *어떻게* 시스템을 만나는가

### Scene 1. 점찍은 아파트 하나, 검증할 방법이 없는 민준

> 평일 저녁 오피스텔. 노트북 앞. 며칠 뒤지고 비교한 끝에 한 단지가 눈에 밟힘. 회사에서 1시간 거리, 24평형, 매매가는 빠듯하지만 대출까지 끌어모으면 닿을 것 같음.
> 
> *"여기… 진짜 괜찮은 것 같은데."*
> 
> 그러다 멈칫. 가격은 적정한가. 학군이 좋다는데 진짜 좋은 건가. 역세권이라는데 도보 몇 분일까. 신축 호재 옆 단지가 들어서면 가치는 오를까 떨어질까.
> 
> 핸드폰을 들었다 놨다. *"한두 사람 말만 듣고 결정할 수는 없다. 그렇다고 처음부터 다 검증하기엔 시간도 지식도 없다."*

### Scene 2. 시스템과의 첫 만남

> "당신이 점찍은 아파트, 사기 전에 다섯 명의 전문가에게 검토받아 보세요."
> 
> *시작하기 →*

### Scene 3. 인터뷰 → 분석 → 토론 → 결정

1. 매물 주소 입력 + 회사 위치·우선순위 (3~4 질문)
2. 5장 카드 객관 점수 산출 (시세·입지·생활편의·미래가치·리스크)
3. 가중치 결정 인터뷰 (5문항)
4. 3 에이전트 가중치 추천 토론 (옹호·반대·중립)
5. 사용자 가중치 직접 조정 → 종합 점수 실시간 변화
6. 최종 점수 + 결정 가이드

> *"아, 이 정도면 결정할 수 있겠다."* — 다섯 명의 전문가 + 본인 가치관으로 검증한 결과를 손에 쥔 상태.

## 7. 시스템이 약속하는 *변화*

| 미사용 시 | 사용 후 |
|---|---|
| 누구 말 믿을지 모름 | 객관 데이터 기반 5개 도메인 점수 |
| 영업 멘트에 휘둘림 | "도보 5분 역세권" 검증 vs 실제 |
| 매물 검증 처음부터 시작 | 단지·인근 데이터 자동 종합 |
| 미래가치 막연한 불안 | 호재 의존도 N% 정량 + 시나리오 분기 |
| 의사결정 무한 미룸 | *자기 우선순위 자각* + 종합 점수 |

> 페르소나 약속: *"AI는 결정 대신해주지 않는다. 사용자를 더 똑똑하게 만든다."*
