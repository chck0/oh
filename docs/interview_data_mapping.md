# 인터뷰 답변 → 데이터 API 활성화 매핑

부동산 검증 AI 에이전트 — 4 질문 인터뷰의 *각 답변 클릭*이 어떤 공공/RAG API를 *어떤 강도로 활성화*하는지의 통합 추적표

기준일: 2026-05-09

---

## 한 줄 요약

> 사용자 *답변 한 클릭* → *세부 항목 가중치* → *API 호출 강도*. 본 문서는 이 3단 체인을 한 자리에서 추적할 수 있도록 모은다. *룰 기반(LLM 0%)* 시스템의 *투명성*을 보여주는 핵심 문서.

[Q × 답변 클릭]

본 문서는 가운데 두 줄 (답변 → 가중치 → API)을 *한 줄로 단축* 한 인덱스.

**총 호출 API** = *답변 무관 13개* + *답변 트리거 7개* = **약 20개** (단지·POI 데이터 포함 캐시 활용).

**활성화 강도 분포**:
- 🔥 강함: 0개 (학군·환경 페르소나 아님 — 1d/1c였다면 8~10개)
- ▲ 단일: 7개
- ✓ 카드 비중: 시세 25% / 미래가치 20% (균형형)

→ 민준 페르소나는 *통근·생활편의 강조* 균형형. 객관 ★3.6 → 가치관 ★3.7 (+0.1)의 *작은 차이*는 이 활성화 강도 분포에서 비롯.
