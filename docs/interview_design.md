부동산 검증 AI 에이전트 — 4 질문 인터뷰 + 답변별 가중치 매핑 룰

기준일: 2026-05-10

---

## 한 줄 요약

> **사용자 가치관을 4 질문으로 측정**하여 47개 세부 항목의 가중치를 결정. 답변 → 가중치 매핑은 *룰 매트릭스* (LLM 0%). *사회적 바람직성 편향*을 회피하기 위해 *직접 중요도*를 묻지 않고 *상황·맥락*을 묻는다.

## 1. 인터뷰 흐름 (Stage 2~4)

```
[Stage 1] 매물 분석 (객관) — 39개 세부 항목 0~5점
    ↓
[Stage 2] 인터뷰 — 4 질문 (3분 이내)
    ↓
[Stage 3] 답 → 세부 항목 가중치 (룰 매트릭스)
    · 답변 → Layer 1 가중치 (1.0 / 0.5 / 0.0)
    · Layer 2 비중 분배 (vector 합산)
    · Layer 3 카드 단위 가중치 (Q4)
    ↓
[Stage 4] 3 에이전트 가중치 토론
    · 옹호: 점수 *최고화* 가중치 조정
    · 중립: 사용자 답 *그대로*
    · 반대: 점수 *최저화* 가중치 조정
```

→ 자세한 런타임 흐름은 [`execution_process.md`](execution_process.md), 세부 항목 정의는 [`sub_items.md`](sub_items.md).

## 2. 설계 원칙 — *왜* 이런 질문 형태인가

### 2.1. 사회적 바람직성 편향 회피

> *"학교가 중요한가요?"* 라고 직접 물으면 한국 부모는 거의 모두 *"중요하다"* 답함. 변별력 ↓.

대신 *상황·맥락 질문*: 자녀 학년·동거 가족 → 학교 가중치 *추론*.

### 2.2. 컨텍스트 vs 가치관 질문 분리

| 종류 | 질문 | 결정하는 것 |
|---|---|---|
| **컨텍스트 (사실)** | Q2 회사 주소 + 통근 | 객관 점수의 *입력 변수* (출퇴근 시간 계산) |
| **가치관 (선호)** | Q1·Q3·Q4 | *가중치* 결정 (점수 산출은 영향 X) |

### 2.3. 변별력 vs 기본 인프라

| 종류 | 인터뷰 묻기? | 객관 점수 |
|---|---|---|
| **변별력 있음** (학교·반려·여가) | ✅ 사람마다 다름 | 일반 산출 |
| **기본 인프라** (24시간 시설·평당가 적정·환원율) | ❌ 모두에게 기본 | *없을 때 큰 감점* (비대칭) |

→ *편의점이 없으면 처음부터 점수가 깎여야* 한다. 사용자에게 *중요?* 묻지 않을 뿐.

### 2.4. 4 질문의 역할 분리 원칙

각 질문은 *서로 영역이 겹치지 않도록* 설계. 한 영역을 두 질문이 동시에 담당하면 가중치 이중 계산 위험.

| 질문 | 역할 유형 | 담당 영역 | 연결 세부 항목 |
|---|---|---|---|
| **Q1 가족 구성** | 가치관 (라이프스테이지) | 학교·의료·환경·치안 | 2.4, 2.5, 3.3~3.6, 3.12, 4.6, 5.1~5.10 |
| **Q2 통근** | 컨텍스트 + 가치관 | 교통 접근성 | 2.1~2.3, 2.6 |
| **Q3 생활 인프라** | 가치관 (직접 선택) | 생활편의 여가 항목 | 3.1, 3.2, 3.7, 3.8, 3.13~3.18 |
| **Q4 보유 기간** | 가치관 (시간 지평) | 시세·미래가치 비중 | 1.3, 1.4, 4.5, 4.6, 4.8, 5.9, L3 |

**분리 결정 근거**:
- **도서관 (3.12)** → Q3 아닌 Q1 연계. 이용하는 도서관 유형이 가족 연령에 의존 (영유아→어린이, 초등→어린이+작은, 중·고·1인→공공).
- **마트 (3.1)** → Q3 담당. 주말 장보기는 생활 여가 패턴의 일부.
- **따릉이 보너스** → Q3 옵션 아님. 시설 유무 자체가 입지 인프라 지표 → 비대칭 보너스로 객관 처리 (서울 한정, 입지 카드 +0.1).
- **시세 (1.1, 1.2)** → 어느 질문도 담당 안 함. 모두에게 동일 기본 지표.

## 3. 답변 → 가중치 표준 매핑

기본 3단 (단일/다중 선택). 일부 특수 조건에서 **2.0 (강조)** 사용.

| 답변 (또는 체크) | L1 가중치 | 적용 조건 |
|---|---|---|
| **강조** | **2.0** | 특수 조건만 — Q2 2b-4 도보 통근 시 2.6 출퇴근 항목 |
| **중요** | 1.0 | 체크함, 해당 답변 선택 |
| **보통** (default) | 0.5 | 인터뷰 답 없거나 중립 |
| **안 중요** | 0.0 | 체크 안 함, 명시적 부정 답변 |

기본값 (인터뷰 답 없거나 영향 없는 세부 항목): **0.5 (보통)**.

> **2.0 사용 원칙**: 도보 통근처럼 *해당 세부 항목이 이미 최고점(★5)에 도달할 가능성이 높고*, 이를 추가로 증폭해야 적합한 경우에 한정. 남용 시 다른 세부 항목과 비중 균형이 깨짐.
