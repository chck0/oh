## 시스템 한눈에

### 5장 카드

| # | 카드 | 핵심 메시지 | 별점 |
|---|---|---|---|
| 1 | 시세 | "이 호가, 적정한가?" | 회귀 잔차·환원율 |
| 2 | 입지 | "영업 멘트가 진짜인가?" | 도보 거리·환승·학교 |
| 3 | 생활편의 | "동네에서 살 만한가?" | 인프라 인접도 |
| 4 | 미래가치 | "5~10년 뒤 가치는?" | 호재 의존도·정비사업·인구 |
| 5 | 리스크 | "이 단지에서 살 때 안정성은?" | 노후·환경·소음·표고 |

### 사용자 흐름 (7 Stage)

```
1. 매물 식별         → 주소 표준화 → HSMP_INNB 매핑
2. 기본 인터뷰       → 회사 위치·가족·우선순위 (3~4 질문)
3. 5장 카드 분석     → 병렬 처리, 별점 산출 (~1분)
4. 가중치 인터뷰     → 5문항 (보유기간·학군·우선순위 등)
5. 3 에이전트 토론   → 옹호·반대·중립 가중치 추천
6. 사용자 조정       → 슬라이더로 가중치 직접 조정
7. 종합 점수 + 가이드 → 최종 결과
```

총 사용자 시간: **약 5분**

## 핵심 가치 (요약)

1. **AI는 결정 대신해주지 않는다** — 사용자를 더 똑똑하게
2. **공공 데이터만** — 직방·호갱노노·리파인 의존 0%
3. **개인정보 최소화** — 자산·재무 안 받음, 매물 객관 검증만
4. **LLM 할루시네이션 0%** — 모든 점수·룰은 데이터에서 자동 산출
5. **SLC** — 시연 시점 모든 카드 완결 작동, "추후 검증" 약속 없음
6. **데이터로 못 잡는 건 아웃풋에 안 담는다** — 약속을 줄여서 약속을 지킴

→ 자세히: [`manifesto.md`](manifesto.md)

## 의사결정 이력 — 핵심 결정만

### ✅ 받아들임

- 5장 카드 (시세·입지·생활편의·미래가치·리스크)
- 사용자 가중치 토론 (인터뷰 → 3 에이전트 → 사용자 조정)
- 트리거 기반 적응형 정보 깊이
- 공공 OpenAPI + RAG 데이터만 (총 55개)
- 미세먼지 *상대값* (시 평균 대비 국지 초과 비율)
- 호재 의존도 분해 (회귀 잔차 + 이벤트 스터디)
- 단지 마스터 키 `HSMP_INNB` 기반 join

### ❌ 거부

- 재무 카드 (개인정보), 법무 자문 (법적 책임)
- 자연재해 카드 (도시 매물 의미 약함), 직방·호갱노노·리파인 (민간 플랫폼)
- LLM 토론 논리 생성 (할루시네이션), 매물 호 단위 검증 (임장 위임)
- 호재 실현 예언 (페르소나 가치 충돌), 5~10년 단일 가격 예측

→ 전체 이력: [`manifesto.md` §의사결정 이력](manifesto.md)
