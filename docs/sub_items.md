# 영역별 평가 점수 — 산출 데이터 정의

부동산 검증 AI 에이전트 — *5장 카드 평가 점수 뒷단에 들어가는 데이터* 매핑

기준일: 2026-05-10

---

## 한 줄 요약

> **영역별 평가 점수 (시세·입지·생활편의·미래가치·리스크 5장 카드)는 *세부 항목들*의 가중평균으로 산출**된다. 본 문서는 *각 카드 점수를 만들어내는 39개 세부 항목과 그 측정 데이터*만 정의. 가중치 적용·인터뷰·debate 로직은 별도 문서·별도 토론.

## 표 컬럼 의미

| 컬럼 | 설명 |
|---|---|
| **#** | 세부 항목 식별자 (예: 2.4 = 입지 카드의 4번째 세부 항목) |
| **세부 항목** | 이름 |
| **무엇을 측정** | 한 줄 정의 |
| **데이터** | 사용하는 공공 OpenAPI / RAG 번호. 클릭하면 [`data_mapping.md`](data_mapping.md) 해당 카테고리로 이동 |
| **기본 비중** | 카드 내에서의 *기본 비중* (모든 사용자 동일). 사용자 가치관 적용 *전* 값 |
